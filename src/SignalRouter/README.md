# SignalRouter UPM Package

Package ID: `com.dreamingdog0529.signalrouter`  
Version: `0.1.0`

This foundation release defines the `SignalRouter.Core` and
`SignalRouter.Protocol` assembly boundaries. It intentionally contains no public
dispatcher, command, result, or registry API yet.

Runtime sources are kept compatible with C# 9. Package consumers do not need to enable
preview language features.
